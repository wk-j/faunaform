# Agent skills

### Issue tracker

Issues and PRDs live as GitHub Issues in this repo (external PRs are **not** a triage surface). See `docs/agents/issue-tracker.md`.

### Triage labels

Five canonical roles (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`); label strings match the role names. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context repo — one `CONTEXT.md` at the root, ADRs under `docs/adr/`. See `docs/agents/domain.md`.
