# แหล่งความรู้: เสียง & Audio Visualization

## Knowledge

- [Seeing Circles, Sines, and Signals — Jack Schaedler](https://jackschaedler.github.io/circles-sines-signals/)
  คู่มือ DSP แบบโต้ตอบได้ อธิบายทุกอย่างด้วยภาพเคลื่อนไหวและเสียง ไม่เน้นคณิตศาสตร์หนัก ๆ — **แหล่งหลักของเรา** ใช้สำหรับ: คลื่น, sine wave, sampling, Fourier Transform/DFT. เริ่มที่หน้า [Introduction](https://jackschaedler.github.io/circles-sines-signals/) แล้วไป [The Fourier Transform](https://jackschaedler.github.io/circles-sines-signals/dft_introduction.html)
- [Visualizations with Web Audio API — MDN](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API/Visualizations_with_Web_Audio_API)
  คู่มือทางการของการวาดภาพจากเสียงด้วย `AnalyserNode`. ใช้สำหรับ: เชื่อมทฤษฎีกับโค้ดที่มีอยู่ใน `js/audio.js`
- [AnalyserNode — MDN](https://developer.mozilla.org/en-US/docs/Web/API/AnalyserNode) ·
  [getByteFrequencyData()](https://developer.mozilla.org/en-US/docs/Web/API/AnalyserNode/getByteFrequencyData) ·
  [frequencyBinCount](https://developer.mozilla.org/en-US/docs/Web/API/AnalyserNode/frequencyBinCount)
  เอกสารอ้างอิงพารามิเตอร์แต่ละตัวที่ Faunaform ใช้ (`fftSize`, `minDecibels`, `smoothingTimeConstant`). ใช้สำหรับ: ตอนจูนค่า

## Wisdom (Communities)

- ยังไม่ได้เสนอ community — จะหาเมื่อผู้เรียนเริ่มมีผลงาน signature ให้คนอื่นวิจารณ์ได้
  (ตัวเลือกที่น่าจะดี: r/DSP, r/webaudio, ฟอรัม Web Audio. ยังไม่ยืนยันคุณภาพ)
