# Faunaform

Faunaform is a browser-based sound visualizer for turning live microphone input into readable, expressive graph forms.

## Language

**Live sound visualizer**:
The main experience that turns microphone input into animated visual graphs in real time.
_Avoid_: Animal detector, classifier, recorder

**Graph form**:
A visual interpretation of the same sound signal, such as spectrum, waveform, or radial energy.
_Avoid_: Animal drawing, mascot, chart decoration

**Scientific creative**:
The visual direction: a precise instrument-like interface with enough motion, color, and light to feel performative.
_Avoid_: Minimal lab, neon performance

**Signature**:
A frozen, loudness-normalized snapshot of a sound's spectral shape, captured over a short window and overlaid against other signatures so two sources read as visibly different. It compares shapes; it does not name or identify the source.
_Avoid_: Animal classifier, species label, fingerprint match, detector
